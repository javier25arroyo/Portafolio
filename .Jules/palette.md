## 2025-05-22 - [Language Switcher Accessibility]
**Learning:** In multi-language sites, even small UI toggles and labels need precise ARIA labels that are also localized to the current user's language. Converting non-semantic interactive spans to buttons is a quick win for keyboard accessibility.
**Action:** Always check if language labels are focusable and if their labels help the user understand what the *next* state will be.
